# Cloud Lab Group Project

A simple static website concept for a campus lost-and-found portal.

## Required Pages

- `index.html`
- `about.html`
- `contact.html`

## Folder Structure

```text
group-website/
├── index.html
├── about.html
├── contact.html
├── style.css
└── images/
    └── image.svg
```

## What to Replace

- Group name
- Course title
- All group member names and roles
- GitHub Pages URL after deployment

The HTML pages describe the project as a lost-and-found portal for the school campus. Replace
the placeholder names with your group's real details before submitting.

## Deploy to GitHub Pages

1. Create a GitHub repository named `cloud-lab-group-project`.
2. Upload these files to the repository.
3. Go to repository **Settings**.
4. Open **Pages**.
5. Select the branch that contains the site files, usually `main`.
6. Set the folder to `/root` and save.
7. Wait for GitHub Pages to publish the site.

## Notes

This project uses plain HTML, CSS, and JavaScript and should work as a static GitHub Pages site.
